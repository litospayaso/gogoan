1.\tLição (1ª Lección)

Esta variante de gallego consta de 22 letras y ocho digrafos. No contiene las letras <k>, <w> e <y> del castellano. La pronunciación de cada letra es la siguiente:;

Letra Sonido Ejemplo
< a > Se pronuncia como una /a/ en castellano.
< b > Se pronuncia como una /b/.
< c > Se pronuncia como una /k/ delante de /a/ /o/ /u/.
Se pronuncia como /Θ/ delante de /e/ /i/. También puede ser pronunciado /s/. 		Casa, cousa, classe, cento, recebe.
< ç > Solo se utiliza con de <a> <o> <u>.
Se pronuncia como /Θ/ o /s/. 		Lição.
< d > Se pronuncia como una /d/.
< e > Se pronuncia como /ε/ o /e/ en posición tónica.
< e > Se pronuncia como /ε/ o /e/ en posición tónica. En posición átona la vocal abierta no aparece.
< f > Se pronuncia como una /f/.
< g > Se pronuncia como /g/ delante de consonante con las vocales <a> <o> <u>.
Se pronuncia como /∫/ con las vocales <e> <i>.
